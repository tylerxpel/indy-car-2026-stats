# XPEL Dealer Quiz Results — June 2026

Interactive report for the XPEL dealer knowledge quiz run via Riddle, covering the campaign period **June 16–24, 2026**.

## Live Page

Published via GitHub Pages: `https://tylerxpel.github.io/indy-car-2026-stats/`

## Contents

| File | Description |
|------|-------------|
| `index.html` | Self-contained report page (styles + charts inline) |
| `.github/workflows/pages.yml` | GitHub Actions workflow that deploys to Pages on push to `main` |

## Enabling GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under *Source*, select **GitHub Actions**.
3. Push to `main` — the workflow deploys automatically.

## Report Highlights

- **415** total quiz views · **Jun 16–24, 2026**
- **246** leads captured (**96.1%** of starters converted)
- **91%** completion rate · avg. **66 s** on quiz
